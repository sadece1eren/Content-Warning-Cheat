can hide and make the menu visible by pressing esc and pressing insert

## Installation

1. Use any mono injector; I recommend Sharp Mono Injector.
2. Select the following options:
   - Namespace: ContentWarningCheat
   - Class Name: Loader
   - Method Name: Init
